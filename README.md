# 🧢 Vida Loka Wear

Loja virtual de roupas para skatistas com estilo urbano inspirado no som e na atitude de Charlie Brown Jr.!

## 🎯 Stack

- Node.js + Express
- HTML/CSS/JS puro
- Catálogo simulado com JSON
